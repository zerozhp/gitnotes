年轻人已经彻底没有机会了？马云反驳：停止抱怨 这就是机会

10月25日，马云现身以色列创新中心开幕式上，其在大会上进行了发言，除了鼓励创新者外，还建议年轻人少停止抱怨发牢骚表示没机会。

马云首先在大会是上阐述了自己对创新者的看法，这部分人可能看上去疯子，骗子，懒惰，靠不住，但是他们内心想要把事情做成的信息和态度都是坚决的，而他本人曾经就是被外人当作疯子来看。

马云在演讲中回忆了自己刚创业的时候，那时的阿里巴巴刚开始起步，当他去硅谷见30个投资者时，把自己把梦想和发展技术说给他们听，大家无一例外的认为马云是疯了，因为电子商务不会在中国成功。也正是靠着马云的加持和对创协的不松懈，阿里巴巴最终成功了，也让西方认识到了电子商务在中国是多么有市场。

马云在发言中强调，自己曾说过中国互联网规模会超过美国，因为中国人口比美国多这么多，当时不少投资者对此都表示怀疑，不过等他们来到国内实地考察后走的时候说，马云我知道你疯了，没想到还能找到100个和你一样疯的。跟美国的电子商务服务大企业，帮他们减少成本不同，马云依然认为阿里巴巴要坚持帮助小企业，帮他们增加收入。

此外，马云还谈到了当下年轻人抱怨没有机会这件事，这事他不认同，只是强调，如果有人在抱怨，这就是机会，因为创新就是为了解决抱怨和焦虑，创造更加美好的生活。

这已经不是马云第一次对年轻人抱怨发声了，之前他的经典语录中就曾提到，自己感谢无数人的抱怨，因为在别人抱怨的时候，才是你的机会，你需要的持续的努力和对做事情的专注度。

上帝对每个人都是公平的，给你关闭一扇窗的同时，又悄悄的打开了另外一扇窗。当你做有觉得机会的事时，在别人抱怨时，你更应该冷静的去看待问题、解决问题，寻求最后的成功，就像马云当年千辛万苦把淘宝做大一样。

之前马云的另一句经典语录也是备受年轻人追捧，人还是要有梦想的，万一实现了呢？虽然看似普通的一句话，却无形中给人不少奋发向上力量，与其抱怨发牢骚，不如保持自己的初心继续努力，这恐怕也是马云说这些话的初衷。
